# Canonical Blender model candidate v1

This snapshot is the Phase 1 identity and rig candidate, not runtime-approved art.

- `neutral.png`: full-resolution transparent three-quarter render.
- `identity-turnaround.png`: front, three-quarter, and profile identity review.
- `scale-qa.png`: readability at 0.55×, 0.75×, 1×, and 1.25×.
- `background-qa.png`: transparent-edge review on four backgrounds.
- `contact-sheet.png`: the twelve-frame Idle identity lock.
- `idle-preview.gif`: slow six-FPS loop preview.
- `qa.json`: alpha, registration, fur-system, rig, lighting, and laptop validation.
- `visual-qa.txt`: independent visual review result.

The complete frames are reproduced with `tools/blender/render_idle.sh`; scratch masters
remain ignored under `assets/renders/work/`.
